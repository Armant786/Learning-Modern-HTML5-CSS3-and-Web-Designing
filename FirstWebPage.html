The Basic Language of the Web:HTML

All morder websites and web application are built using three fundamental technologies: HTML,, CSS and Javascript. Thse are the language of the web.

In this post, let's focus on HTML. we will learn what HTML is all about, nad why you too should learn it.

What Is HTML?

HTML stand for HyperText Markup Languasge .It's a markup use language that web developers use to structure and describe the content of a webpage (not a programing language).

HTML consists of elements that describe deffernt types of content: paragraphs, links, headings, images, video, etc. Web browsers understand HTML and render HTML code as websites.

In HTML, each element is made up of 3 parts:

  1. First
  2. The Opening tag
  3. The Closing tag
  4. The Actual element

Why Should you learn HTML?

Ther are countless reasons for learning the fundamentals the fundamentals language of the web, Here are 5 of them:

  • To be abale to use the fundamentals web dev language
  • To hand-craft beautifulwebsites instead of relying on tools like Wopress or Mix
  • To build web applications
  • To impress friends
  • To have fun
